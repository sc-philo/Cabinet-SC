1. Préambule

La présente politique de confidentialité a pour objet d’informer les visiteurs et usagers du site sur la manière dont leurs données personnelles sont collectées, traitées et protégées. 3. Données collectées

Données directes :

  • Nom, prénom
  • Email
  • Numéro de téléphone
  • Informations de paiement (gérées exclusivement par Stripe)
  • Date et heure de réservation

Données automatiques :

  • Adresse IP
  • Données de navigation via cookies (strictement nécessaires et statistiques anonymes)

4. Finalités du traitement

Les données sont traitées pour :

  • Gérer les rendez-vous
  • Communiquer avec les clients
  • Gérer la facturation via Stripe
  • Améliorer l’expérience utilisateur

5. Destinataires des données

Les données sont destinées uniquement :

  • À Sarah Cohen
  • À Stripe pour le traitement sécurisé des paiements

6. Durée de conservation

  • Données de réservation : 3 ans
  • Données de facturation : 10 ans
  • Cookies : 13 mois

Conformément au RGPD, vous pouvez :

Sarah Cohen prend toutes les mesures nécessaires pour protéger vos données. Les paiements sont sécurisés via Stripe.
